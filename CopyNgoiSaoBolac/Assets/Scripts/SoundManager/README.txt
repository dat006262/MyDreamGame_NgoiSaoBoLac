Cung cap 1 AudioManager để phát ra Nhạc nền, các nhạc khác
Cách sử dụng:
Gắn nhạc nền vào BGM và trường BgmClip; Volumn điền (min 0, ,max1)
Gắn nhạc Effect thành phần vào SFX Clip và set volumn VD volumn =1(max)

Để chạy nhạc thì tại fileMono chạy dòng lệnh AudioManager.intances.PlaySFX(AudioManager.SFX.HIT);
